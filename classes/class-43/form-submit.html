2️⃣Then we need to createContext and export it.

3️⃣We used the {children} as prop on InstallProvider.

4️⃣And use InstallContext.Provider and wrap it on the {chidren}.

5️⃣ Here we we used useState for app installation. The initial value of useState is an empty array.

6️⃣ We took a function named installApp which had app as prop. Then we setInstallledApps where we used the prev value. We used find for seeing if the installed app already exist or not.
➡️If the app exists then return prev that is, keep the previous list.
➡️If it does not exist the we keep a copy of prev and add the new app that is, [...prev, app].

8️⃣In AppDetails.jsx, we used useContext for the InstallContext. And we destructured installedApps and installApp

9️⃣Then we check whether the app is installed or not using find on the installedApps list.

☑️Uninstall App (deleting)

1️⃣Here, in InstallContext.jsx, we took a uninstallApp function with an id as prop. There we update the installed apps using setInstalledApps and from the prev (current list before the update) installed app list filter using id. So, it only keeps the app that are not equal to the id matched.

2️⃣In InstallContext.jsx, we pass the uninstallApp as value in the InstallContext.Provider.

3️⃣In Installed.jsx, we use the useContext and destructure uninstallApp.

4️⃣Here we used onClick on the uninstall button and call the uninstallApp function and pass the app id.
