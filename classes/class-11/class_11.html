✔️How to get Started With Tailwind CSS

At first we need to go to tailwindcss.com and from their Play CDN copy the script part and paste it in the head.
Then, we can install the Tailwind CSS intelliSense extension on our VS code. We need to create a .css file and paste @import "tailwindcss"; on it to make the extension to give suggestions.

✔️Where to write Tailwind CSS?

Tailwind is written as classes inside the elements. No separate CSS file is needed BUT it can be VERY class heavy and harder to work with.
