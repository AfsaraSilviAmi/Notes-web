✔️What is ES6

ES6 (ECMAScript 2015) is the 6th version of JavaScript that introduced modern features to make JS cleaner, more readable,more powerful and better for large applications.

✔️Difference between var, const and let

✅Var:

Here, if we want then we can use console.log(Name); before declaring the variable. It will give undefined. So, it is not block-scoped. And using var when reassigning is allowed. This cas cause a lot of bugs. It's better not to use it.

✅Let:

Here, if we use console.log(Name); before declaring the variable, it will give an error. So, it is block-scoped. And using let (redeclaring) when reassigning is not allowed. It should be used when we want to change the value of the variable.

✅const:

It is block-scoped. And it can not be reassigned.

But constant object can be modified.

✔️Rest Operator

The rest parameter (...) allows a function to accept any number of arguments and collect them into a single array.

✔️Spread Operator

Used to expand arrays or objects into individual elements. When copying using spread operator, we don't copy the WHOLE array like we usually do. Instead, we copy the elements within that array. Thus changing the 1st array doesn't change the 2nd one.

✔️Object and Array Destructuring

Destructuring allows us to extract values from arrays or properties from objects and store them into variables.

✅Array Destructuring:

We are able to SKIP values by giving gap for the values we don't want.

✅Object Destructuring:
