✔️Find the Word Frequency and Return as Object

1. At first, in function, we split() the string into array.
2. Then used toLowerCase() method. So, that cases are ignored.
3. Then used for of loop to go through the elements one by one.
4. The count = {} is an empty object where all the result/word counts will be stored.
5. Then we used count.hasOwnProperty(word) in if condition. It checks the words stored in count has the same property name or not.
6. If it has the same property, we make count[word]++. Here, count is the object and word is the key/property. So, count[word] is a way to access the value.
7. Else, the count[word] = 1.
8. And by returning the count we get the desired output.
