✔️Anchor Tag

Using the anchor tag <a> I am able to go to the Previous page and from that page by clicking the Next page button I can come back to this page.
The anchor tag is used by putting the hyperlink reference (href) inside the anchor tag.

✅And to make sure that the link opens in a new tab use target = "_blank" inside the anchor tag.

✔️Images tag

We use img tag for images. Image tag has src attribute to show the image.


✔️Form making

For making a form we need A label tag, a input tag, select tag, option tag.
