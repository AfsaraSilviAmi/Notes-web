Class 6- Pusedo and Position

✔️Psuedo

  1. Hover (Ex-li:hover{color:white;}).
  2. focus(is used pn input).
  3. visited(can be used in case of anchor).
  4. last-child (can be used to change the CSS for the last option os the list...Ex-li:last-child{color:red;}).
  5. first-child(same as last child but used for the 1st option).
  6. nth-child(2n)--> Used for Even option.
  7. nth-child(2n+1)--> Used for Odd option. And so on...can be used for 3n and so.
