✔️Types of operators

  1. Arithmetic operator: (addition: + , substitution: -, multiplication: * , division: / , modulus operator: %)
  2. Comparison operator: (==, ===, > , < , >=, <=, !=, !==)
  3. Logical operator: (&&, ||, !)

✔️Arithmetic operators

let ict = 85;
let bangla = 65;

console.log(bangla + ict);
console.log(bangla - ict);
console.log(bangla * ict);
console.log((bangla / ict).toFixed(2));
console.log(13 % 2);

✔️Comparison operators

Comparison operators always get boolean output!!!

let physics = 50;
let math = "50";

console.log(physics == math); (checks only value and thus the output will be TRUE even though one is a number and the other is a string)
console.log(physics === math); (checks for both value and datatype-- so it's output will be FALSE)

let biology = 65;
let ict = 55;

console.log(biology > ict); (true)
console.log(biology>=ict); (true)
console.log(biology < ict); (false)
console.log(biology <= ict); (false)
console.log(biology != ict); (it a loose not equal...that means it DOES NOT check the data type)
console.log(biology !== ict); (it checks both the data type and value)

✔️Logical operators

  • && operator (All the Conditions need to be TRUE for the OUTPUT to be TRUE, otherwise it's false)
  • || operator (Any one of the Conditions need to be TRUE for the OUTPUT to be TRUE, it's FALSE only when all conditions are FALSE)
  • ! operator (reverses the value of true or false)(ex--console.log(!isMatured);)
