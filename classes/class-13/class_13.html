✔️Variable

Dos and Don'ts for Variable names:

  • Letters (a-z, A-Z) can be used.
  • We can use Numbers BUT not at the start.
  • No special characters can be used except Underscore ( _ ).

✔️Variable Naming Conventions

The three variable naming Conventions are:

  1. camelCase (which we will use)
  2. snake_case
  3. PascalCase

✔️var

Var is used to declare a varying variable. But we will use let and const instead since they don't allow declaring variable using same name unlike var.

✔️let

In case of let reassignment is possible.

Example-- let age = 25;
age = 26; // re-assign
age = 27; // re-assign
age = 28; // re-assign
console.log(age);

The output will show the latest assigned value.

✔️const

In case of const reassignment is NOT possible.

Example-- const age = 25;
age = 26; // re-assign
console.log(age);

The output will show ERROR!!
