Data Type

✔️Types of JavaScript

There are 2 types of JavaScript:

  1. Primitive data type (i. string, ii. number, iii. boolean, iv. undefined, v. null)✅
  2. Non primitive or referencial data type (i. array , ii. object)✅

✔️Primitive Data Types

  1. string (let country = "Japan";)
  2. number (let age = 25;)
  3. floating number (let cgpa = 3.81;)
  4. boolean (let isPassed = true)
  5. undefined (let schoolName;)(no value is assigned)
  6. null (let uniName = null;)
