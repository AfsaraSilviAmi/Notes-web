✔️Converting String to Number

Example--

let age = "25"; (here the value is a string)
let age2 = Number(age); (in this method we use the Number()) ✅
let age2 = parseInt(age); (another way to convert is using parseInt())✅

✔️Converting String to Float

Example--

let cgpa = "4.64";(here the value is a string)
console.log(parseFloat(cgpa)); (in this method we use the parseFloat) ✅

let floatSum = 0.1 + 0.2;
console.log(floatSum.toFixed(4)); (this is used to fix the number decimals after the decimal point)

✔️Converting Number to String

Example--
