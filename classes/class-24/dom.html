✔️DOM(Document Object Model)

It treats the HTML like a tree of objects that JS can access and change.

✔️Selecting Elements

  1. ✅getElementById():

    Selects ONE element by Id. And returns a single element. Example---

    html:
    <p id = "demo">Some Text</p>

    js:
    let p = document.getElementById("demo");

  2. ✅getElementsByClassName():

    Selects multiple elements by class. And returns HTML collection. Example---

    html:
    <p class= "hello">Hello World</p>
    <p class = "hello">Hello Universe</p>

    js:
    let words = document.getElementsByClassName("hello");

  3. ✅getElementsByTagName():

    Selects elements by tag Name. And returns HTML collection or multiple elements. Example---

    html:
    <p>One</p>
    <p>Two</p>

    js:
    let p = document.getElementsByTagName("p");

  4. ✅querySelector():

    Selects only first match in case of classes. And returns a single element. For querySelector it is better to use it for id but it can be used for classes as well but it will only picks the 1st element/match. Example---

    js:
    document.querySelector(".hello");

  5. ✅querySelectorAll():

    Selects all matches. And returns a Node List. Example---

    js:
    document.querySelectorAll(".hello");

✔️NodeList Vs HTMLCollection

✔️Changing Content

  1. ✅innerText:

    Changes visible text only. Example---

    js:
    p.innerText = "New Text";

  2. ✅innerHTML:

    Changes HTML inside. Example---

    js:
    section.innerText = "<b>Bold</b>";

✔️Attributes

  1. ✅getAttribute():

    html:
    <img id ="img" src ="pic.png">

    js:
    document.getElementById("img").getAttribute("src");

  2. ✅setAttribute():

    js:
    img.setAttribute("src", "new.jpg");

✔️classList

  1. ✅classList.add():

    js:
    section.classList.add("bold");
