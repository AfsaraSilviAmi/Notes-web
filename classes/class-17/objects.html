Objects

✔️Objects

Objects are non primitive data type. Example---

let bike = {
color: "Black",
brand: "Honda",
isMeterDigital: true,
advantages: ["less milage", "High speed", "Low configuration"],
battery: {
color: "black",
durability: "strong",
warranty: "2 year",
}
};

Here, color,brand etc are keys/properties and Black, Honda etc are values.

✔️Accessing a specific property or key

  1. dot notation . Example-- console.log(bike.color);
  2. bracket notation [ ] Example-- console.log(bike[color]);

Output:Black



✔️Update

Updating an object is possible. Example ---

bike.color = "Lavender";
console.log(bike);

✔️Delete

Using delete a key and it's value is possible. Example ---

delete bike.brand;
console.log(bike);
