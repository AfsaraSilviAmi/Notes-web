✔️What is Array?

Array is used to store more than one values in a single variable.
Example--
let fruits =["apple", "grape", "orange"];

✔️Index

Index of an array starts from 0.
Example---
let roll =[5, 44, 2, 78]
console.log(roll[1]); ---- (output will be 44) ✅
roll[2] = 67; --- (here the third element ,that is, 2 is changed to 67)✅

✔️Array Length

Array length is used to determine the number of elements in an array.
Example---
let roll =[5, 44, 2, 78]
console.log(roll.length);----(here the output would be 4)✅

✔️Push

Push is used to insert an element at the end of an array.
Example--
fruits =["apple", "grape", "orange"];
fruits.push("tomato"); ----(tomato will be inserted at the end of the array fruits)
console.log(fruits)

Output:["apple", "grape", "orange","tomato"]

✔️Pop

Pop is used to remove an element from the end of an array.
Example--
fruits =["apple", "grape", "orange"];
fruits.pop(); ----(orange will be eliminated from the end of the array fruits)
console.log(fruits)

Output:["apple", "grape"]

✔️Shift

Shift is used to remove an element from the Start of an array.
Example--
fruits =["apple", "grape", "orange"];
fruits.shift(); ----(apple will be eliminated as it is the 1st element in the current array)
console.log(fruits)

Output:["grape", "orange"]

✔️Unshift

Unshift is used to add an element to the Start of an array.
Example--
fruits =["apple", "grape", "orange"];
fruits.unshift("mango"); ----(mango will be inserted at the start of the array fruits)
console.log(fruits)

Output:["mango","apple","grape", "orange"]

✔️Includes

Includes is used to find an element in the array. If the element exists in the array then it gives the output true otherwise false.
Example--
fruits =["apple", "grape", "orange"];
fruits.includes("mango");
console.log(fruits)

Output: false

✔️indexOf

indexOf is used to find the index of an element in the array. If the element exists in the array then it gives the output as the index number of that element otherwise it gives -1.
Example--
fruits =["apple", "grape", "orange"];
fruits.indexOf("orange");
console.log(fruits)

Output: 2

✔️isArray

isArray is used to determine whether the variable is an array or not.
Example--
fruits =["apple", "grape", "orange"];
console.log(Array.isArray(fruits))

Output: true

✔️concat

concat is used to combine arrays.
Example--
fruits =["apple", "grape", "orange"];
let roll =[5, 44, 2, 78]
console.log(fruits.concat(roll))

Output: ["apple", "grape", "orange",5, 44, 2, 78]
