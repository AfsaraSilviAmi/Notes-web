✔️Installing Tailwind in React

1️⃣At first, make a vite project. And then go to tailwind website documentation.

2️⃣Paste the npm intall tailwindcss @tailwindcss/vite on the VScode bash terminal.

3️⃣Paste the necessary plugins as instructed at the document in the vite.config.js.

4️⃣We need to import the taiwind as instructed in thr document. We paste it in any CSS file(such as-- app.css pr index.css)

⚠️⚠️In case tailwind intellisense DO NOT WORK, then just click on Cntrl + spacebar.

✔️Installing DaisyUI in React

1️⃣Go to DaisyUI website. Start by pasting npm i-D daisyui@latest in the VScode terminal.

2️⃣Then, in CSS file(such as-- app.css or index.css) paste @plugin "daisyui"; as instructed in the document.

✔️Installing Icons in React

For icons we can go to lucid icons and follow the installation guide. We must choose the npm one. Then we can search the icon. ⚠️Remember to import each icons...usually the extension helps to do so.

We can install react icons for using icons.

✔️React Extension (React Extension Pack)

1️⃣Installing the React Extension Pack in VScode is a must. Can be searched through the VScode extension.

2️⃣After making a new .jsx file, write rsc to get a component shortcut.

3️⃣This extension also helps to import automatically.

⚠️⚠️ JSON file (ex--file.json) must be saved in the public folder.

✔️Installing Recharts

Reacharts are used to make graphs and charts for the json data.

1️⃣Go to Reacharts websites, install according to guide(npm). Follow the documentation to make graphs, pie charts etc.

✅Tooltip is useful for showing data when hovering the graph.

✔️Installing Toastify

Toastify is used to event notification/alert in a beautiful look.

1️⃣Go to React Toastify website, and according to documentaion install it.

2️⃣Paste npm i react-toastify on the VScode terminal.

3️⃣Paste <ToastContainer /> at the end but inside the component of app.jsx or paste it in index.html. ⚠️Make sure the extension imports it!!
