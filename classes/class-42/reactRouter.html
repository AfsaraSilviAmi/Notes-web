✔️createBrowserRouter

We can make routes by createBrowserRouter. It is an array that can have multiple objects. Each object at least has 2 properties--- path and element/component.

In case of element---

{
path:'app'
element: <App></App>
}

In case of component---

{
path: 'app2'
Component: AppTwo
}
