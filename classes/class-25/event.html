✔️Mouse Events

  • 1️⃣mouseenter

    mouseenter happens when the mouse pointer enters an element. Example--

    let box = document.getElementById("box");
    box.addEventListener("mouseenter", function(){
    console.log("Mouse entered the box");
    });

  • 2️⃣mousemove

    mousemove runs every time the mouse moves inside an element.

  • 3️⃣mouseout

    mouseout happens when the mouse leaves the element.

✔️Input Field Events

  • 1️⃣focus

    focus happens when an input field becomes active (clicked or selected). Example---

    let input = document.getElementById("name")
    input.addEventListener("focus",()=>{
    console.log("The input is selected")
    })

  • 2️⃣blur

    blur happens when the input field looses focus.

✔️Keyboard Event

  • 1️⃣keydown

    keydown happens when a key is pressed on the keyboard. Example---

    input.addEventListener("keydown", function(){
    console.log("Key pressed");
    });

  • 2️⃣keyup

    keyup runs when a key is released.

✔️event

event is an object that stores information about the event (like which key was pressed). Example---

makeDiv.addEventListener("click", function(event){
console.log(event);
});

✔️Event Bubbling

When an event happens on an element, it “bubbles up” the DOM tree, triggering the same event on it's parent element unless it's stopped.

✔️Event Delegation

Instead of adding event listeners to every child, we add one listener to a parent and handle the event using event.target.
