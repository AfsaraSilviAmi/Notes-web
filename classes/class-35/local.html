✔️Local Storage

LocalStorage is a browser storage that -- Stores data permanently (until it is cleared), saves data as key-value pairs and can only store strings.

🔍How to view Local Storage in Browser

1️⃣Right click and then click Inspect

2️⃣Go to the Application tab.

3️⃣Click on Local Storage

4️⃣Select your site (key-value pairs might be seen)

✔️localStorage.setItem()

localStorage.setItem() is used to save data. Example---

localStorage.setItem("name", "Ami");

✔️localStorage.getItem()

localStorage.getItem() is used to get the data. Example---

localStorage.getItem("name");

console.log(name);

Output: Ami

✔️localStorage.removeItem()

localStorage.removeItem() is used to remove one item. Example---

localStorage.removeItem("name");

✔️localStorage.clear()

localStorage.clear() is used to clear Everything. Example---

localStorage.clear();

⚠️Only Stores Strings!!

In order to store object or array we must convert it.

✅Store object/array

const user = { name: "Ami", age: 24 };

localStorage.setItem("user", JSON.stringify(user)); //converts object or array into string using stringify

✅Get object/array back

const data = localStorage.getItem("user");

const user = JSON.parse(data); //converts string into object using parse

console.log(user.name);

Output: Ami
