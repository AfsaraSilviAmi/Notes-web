✔️React

React is a JavaScript library for building UI (not a full framework). React is reusable that is, we can make many components(NavBar, Banner etc) in react which can be reused.

Library is something we call when we need it.
Framework has it's own rules and we must follow it.

✔️React Vs Next.js Vs Vue

Feature React Next.js Vue
Type Library (like a toolbox where one can pick their desired tool) Framework (like a factory which has it's own building process) Framework
Developer Meta Vercel Evan You
SEO Weak Strong Medium
Routing Manual (React Router) Built-in Built-in (Vue Router)
Performance Good Very Fast Good
Flexibility High Medium Medium
Backend Support No Yes (API routes) No
Use Case SPAs, dashboards SEO websites, full-stack apps Simple apps, fast development

✔️JSX(JavaScript XML)

JSX lets one write HTML-like code inside JavaScript. In react, we use the jsx extension(App.jsx, NavBar.jsx etc).

✔️JSX Rules

✅Must return a single parent that is only One div is accepted inside return. Nested div is accepted.

✅All tags should be closed.

✅We use className instead of class in React.

✅We use CamelCase for attributes.

✅✅We can use JavaScript only INSIDE CURLY BRACKETS.
