✔️How to make a Form(Radio button main)

  1. Using input we can select the radio button.
  2. We can add the label afterwords
  3. Make sure the for in the label is same as the id in the input field. This associates that input for that label.
  4. Now, we want to make it so that we can Select ONLY ONE of the options. In such case the name in the input field for all options MUST BE same.

    5. PS. Use shortcut li*5 for making several li
