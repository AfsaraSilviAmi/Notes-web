✔️Function

Function is a REUSABLE block of code. It may or may not have parameters and the parameters are ONLY available INSIDE the function. But it must contain a function Name. And in { } there should be some executable code. A function may or may not have a return, it depends on the code.

Syntax: function functionName(list of parameters){
function code;
return returnValue;
}

✔️Function call

Function call is used to call the function. A function call must have the function Name. And it may or may not have arguments. The arguments are the values which are sent to the function.

Syntax: functionName(arguments);
