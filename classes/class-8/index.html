✔️Responsive Gallery

Here the Images are Taken in separate divs and is assigned classes based on the space it would take. P.S. NOT ALL OF THE DIVS ARE SHOWN IN THE SS.
In desktop, to make it in a defined layout one has to make the divs grid by writing display:grid on their parent div. And use grid-template-columns and grid-template-rows(if needed) to specify the number of columns and rows. The 1fr determines the fraction of space it will take. We use span on grid-column and grid-rowto determine the space it takes.

✔️Gradient

To make a text gradient we MUST use -webkit-background-clip:text and -webkit-text-fill-color:transparent!!!
