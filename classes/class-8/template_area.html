✔️Template-areas

Above we have the html file.

Here, each of the sections/or the containers we want take definite space MUST BE assigned a grid area of any name. That grid area are to be used to set the areas row-wise in the grid-template-areas. Remember to set the grid area to unset and the grid-template-areas to none if you want your mobile version to not have any specific layout of this sort.
