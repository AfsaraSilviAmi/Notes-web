✔️JSON

JSON stands for JavaScript Object Notation. It is a data format used to store and send data. It looks very similar to JavaScript Objects. Example---

{
"name": "Ami",
"age":25,
"city":"Dhaka"
}

The only difference is that the properties in JSON are inside quotation.

  • 1️⃣JSON.stringify()

    JSON.stringify() converts a JavaScript object → JSON string. Example---

    let user = {
    name: "Ami",
    age:25,
    city:"Dhaka"
    }

    let JSONData = JSON.stringify(user);

    console.log(JSONData);

    Output: {"name": "Ami", "age":25, "city":"Dhaka"}

  • 2️⃣JSON.parse()

    JSON.parse() converts a JSON string → JavaScript object. Example---

    let jsonData = '{"name":"Ami","age":24}';

    let user = JSON.parse(jsonData);

    console.log(user.name);

    Output: Ami

✔️fetch()

The fetch() method is used to fetch the API from the server.

✔️loading and displaying

Using fetch we get the api. After converting it into json format using the json() method, we get the data.

We load the data in a display function in order to display it. Usually involves getting the container id, clearing it's innerHTML, going through a loop to create several divs, using template strings to show the values of the properties in an object and append it to the container.

✔️async/await

✅async makes a function return a promise, ✅await pauses code until the promise is done.
