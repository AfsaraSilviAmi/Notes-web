✔️BetterAuth and Mongodb setup

1️⃣Go to BetterAuth website (follow the documentation) and install BetterAuth using npm install better-auth.

2️⃣Create .env file in the root and copy the better auth secret key and URL. The URL will change base on the deployment link---

3️⃣Create lib folder in src and make a auth.js file.

4️⃣Then, we will install mongodb using npm install mongodb. And if we go the Database section/sidebar of the document, we will get the command for installing mongo adapter using npm install @better-auth/mongo-adapter. After that we will finish the work for auth.js (can be found in the Authentication section/sidebar as well)---

5️⃣Then we will go to the Mongobd Atlas website. And SignUp. Then, we will go to the Database and network sidebar and make a database/user (copy the username and password for later use). Then we will go to the Cluster sidebar and click Connect then click Driver where we will get a link. Copy the link and paste it in .env with a variable name such as- AUTH_DB_URI.
